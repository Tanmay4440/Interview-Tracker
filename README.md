# 🎯 AI Interview Preparation System

The AI Interview Preparation System is an interactive web application built using Streamlit to help students evaluate their technical interview readiness and create a structured preparation strategy.

The platform allows users to assess their proficiency across important computer science subjects such as Arrays, Linked Lists, Stack, Queue, DBMS, and Operating Systems. Based on the assessment scores, the system identifies strong and weak areas, visualizes performance through analytics dashboards, and generates personalized learning roadmaps to guide improvement.

To support interview preparation, the application includes a dynamic interview question generator that provides topic-based practice questions at different difficulty levels. Students can use these questions to strengthen their understanding of core concepts and improve problem-solving skills.

The system also incorporates a machine learning component using a Decision Tree Classifier to predict a student's interview readiness level. Based on the overall performance, users are categorized into readiness levels such as Low, Medium, Good, or Excellent, along with corresponding recommendations.

## Key Features

* Student profile management
* Technical skill assessment across multiple subjects
* Performance analytics dashboard with visualizations
* Weak topic detection and strong topic identification
* Personalized learning roadmap generation
* Topic-wise interview question generation
* Progress tracking system
* Interview readiness prediction using Machine Learning

## Technologies Used

* Python
* Streamlit
* Pandas
* Matplotlib
* Scikit-Learn
* JSON

## Machine Learning Component

The project uses a Decision Tree Classification model to estimate interview readiness based on a student's average performance score. This demonstrates the application of supervised learning techniques in educational analytics and personalized learning systems.

## Objective

The primary objective of this project is to provide students with a structured and data-driven approach to technical interview preparation. By combining analytics, machine learning, and guided learning paths, the system helps users identify gaps in their knowledge and focus their preparation more effectively.
